# Журнал изменений

В этом файле описаны заметные изменения проекта.

## [1.0.0] - 2026-06-22

### Добавлено

- Первая однофайловая TUI-реализация на Python 3.8.
- Одновременный сканер файловой системы с ограниченным пулом потоков.
- Режимы раскладки treemap: wide и square.
- Цветовая кодировка типов файлов и яркость по размеру.
- Интерактивная навигация по каталогам, хлебные крошки, статусная строка и боковая панель.
- Исключение каталогов по имени или пути.
- Исключение NFS/CIFS/SMB и пользовательских типов файловых систем.
- Поддержка терминалов Linux, macOS и Windows без сторонних пакетов.
- README на английском и русском языках.
- Архитектурные диаграммы PlantUML на английском и русском языках.
