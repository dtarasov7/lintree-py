# Журнал изменений

В этом файле описаны заметные изменения проекта.

## [1.2.0] - 2026-07-01

### Добавлено

- Добавлена горячая клавиша `m` для переключения treemap между отображением по занимаемому месту и по количеству файлов.
- Добавлена поддержка активной метрики для раскладки treemap, агрегирования top-элементов, яркости ячеек, подписей, `% Parent` и списка дочерних элементов в боковой панели.
- Добавлены индикатор `m:size` / `m:files` в статусной строке и подсказка в help overlay.

## [1.1.0] - 2026-06-22

### Изменено

- Переработана навигация: стрелки теперь перемещают фокус пространственно, а `j` и `k` идут по визуальному порядку чтения.
- Рамка выбранной ячейки заменена с half-block символов на более понятную псевдографику.
- `--fast` стал основным документированным флагом быстрого сканирования, а `-fast` оставлен как alias для совместимости.
- Добавлены индикаторы режима сканирования на экране прогресса и в статусной строке.
- Обновлены описания управления и возможностей в README.

## [1.0.0] - 2026-06-22

### Добавлено

- Первая однофайловая TUI-реализация на Python 3.8.
- Одновременный сканер файловой системы с ограниченным пулом потоков.
- Режимы раскладки treemap: wide и square.
- Цветовая кодировка типов файлов и яркость по размеру.
- Интерактивная навигация по каталогам, хлебные крошки, статусная строка и боковая панель.
- Исключение каталогов по имени или пути.
- Исключение NFS/CIFS/SMB и пользовательских типов файловых систем.
- Поддержка терминалов Linux, macOS и Windows без сторонних пакетов.
- README на английском и русском языках.
- Архитектурные диаграммы PlantUML на английском и русском языках.
